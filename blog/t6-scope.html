Scope

10 may 18

What is scope?

When trying to describe scope, the most general analogy I can give is how laws are different / applied in different countries, regions, or states. For example, laws that are specific only to New Zealand, cannot be applied or used as precedence in Zimbabwe.

How does it work in JavaScript?

Scope has the same effect in JavaScript: when variables are defined within the "scope" of a function, their scope can be seen as local -- i.e., has effect only within the function itself.

Conversely, when variables are defined outside of a function's "scope", their presence has effects globally -- i.e. throughout the entire program.

Personally, when building the calculator for this sprint's project, I forgot to update the value of a global variable with a certain function's results -- meaning that whenever the global variable was called for reference later, it was the same as it was defined originally. This basically rendered the method's functionality useless.
