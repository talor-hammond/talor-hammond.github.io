JavaScript: Language

13 apr 18

In the JavaScript language, describe all the use cases for each of: parentheses, square brackets, curly braces, single quotes, and double quotes.

Parentheses, ()
Parentheses can be used for grouping and containing expressions, and can also have multiple functions relating to functions and structures. For example, parentheses are used to contain a list of parameters for a specific function -- if the function has no parameters, it can be called with empty parentheses: myFunction().

Square brackets, []
Brackets can be used to reference / access objects from an array through an index -- this is called bracket-notation. The first item in this index can be accessed with '[0]', i.e., the index starts from zero, and not 1.

Curly braces, {}
Curly braces are used to define the beginning and end of function blocks, as well as the if statement and for loop structures.
Additionally, they are used to initialise JavaScript objects - for example:

var object = { propertyOne: "x", propertyTwo: "y" }

Single & double quotes, '' & ""
Single and double quotes can be used interchangeably to define strings, for example:

var myString = "Hello"; or,
var myString = 'Hello';

The key difference with either quotation marks, is that single quotes can be used inside double quotes, without having to escape the quotation marks using backslash.
